## 📌 What is TaskWave?

TaskWave is a modern web app designed to help you keep track of your tasks. It works both online and offline, meaning you can use it anytime without worries about internet connection. TaskWave helps you organize tasks by priority, tag them with labels, filter and sort easily, and reorder your list by dragging items. You also get reminders if tasks are overdue and insights to see how productive you are each week. The app looks smooth and works well on both desktop and mobile devices. You can switch between light and dark themes to suit your mood or environment.

## 🔧 How to Use TaskWave

### Adding Tasks

- Click the **Add Task** button.
- Enter a title for your task.
- Optional: Add a description to give more details.
- Set priority with one click (Low, Medium, High).
- Assign tags by typing keywords.
- Save the task; it appears in your list.

### Managing Tasks

- Drag a task up or down to reorder.
- Click a task to edit details or change status.
- Use the filter bar to show tasks by priority, tag, or due date.
- Mark tasks as complete by clicking the checkbox next to them.
- Overdue tasks are highlighted with a red badge.

### Using Offline Mode

- If you lose internet, TaskWave keeps working.
- The app saves your data locally.
- When you reconnect, it syncs changes automatically.
